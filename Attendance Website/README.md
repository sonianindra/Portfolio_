Attendance Website

Description:
A PHP-MySQL-based web application for tracking fraternity event attendance, calculating points per user based on event participation. The system supports user registration, event creation, and point tracking with secure database integration.

Tech Stack:
Frontend: HTML/CSS
Backend: PHP
Database: MySQL
Hosting: Bluehost (cPanel, phpMyAdmin)

Features:
Event management (event name, category, points, and date)
User attendance submission and validation
Point tracking per user with real-time updates
Schema includes three core tables: users, events, and attendance with foreign key constraints

Usage:
Navigate to the attendance submission form
Select event and confirm attendance
Admin panel supports user and event management
